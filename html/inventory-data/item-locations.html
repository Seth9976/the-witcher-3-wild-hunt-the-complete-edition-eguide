Inventory Data Item Locations

Included is a list of all shop definitions and the items sold at each.

Crafting

All Crafting Components are defined in one list at the top. It includes minimum level requirements, Quantity Range and Spawn Chance which uses -1 to indicate that an item should always be listed. Values above zero indicate the % Chance an Item will be listed.

Blacksmith

Blacksmiths use the shop definition based on the region they are located; White Orchard, Velen (No Man's Land), Novigrad and Skellige. For specific item placement, each Merchant also references a custom shop definition listed below the common includes.

Armorer

Armorers use the shop definition based on the region they are located; White Orchard, Velen (No Man's Land), Novigrad and Skellige. For specific item placement, each Merchant also references a custom shop definition listed below the common includes.

Merchants

All other Merchants types including Herbalists, Alchemists, Tailors, Wandering Merchants, etc. all reference a custom shop definition.

PLEASE NOTE! Certain items may not have the exact name as it appears in the merchant's shop menu, so you are wise to search by Merchant, as well as item, to pinpoint an item you're after.
